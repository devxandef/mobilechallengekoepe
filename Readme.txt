# Desafio Coodesh: Dicionário de Palavras

O **Desafio Coodesh: Dicionário de Palavras** é uma aplicação React Native que permite aos usuários pesquisar palavras, visualizar seus significados e fonética, salvar palavras como favoritas e acessar uma lista com as palavras previamente pesquisadas. Além disso, a aplicação salva em cache os resultados das requisições à API de palavras para agilizar buscas com parâmetros repetidos.

## Tecnologias Utilizadas

- Linguagem: **JavaScript (ES6+)**
- Framework: **React Native**
- Tecnologias adicionais: **Redux, Redux Toolkit, React Navigation, Axios, Firebase (opcional para autenticação), Async Storage**

## Funcionalidades Obrigatórias

1. Visualizar uma lista de palavras com rolagem infinita.
2. Visualizar detalhes de uma palavra, incluindo significados e fonética.
3. Salvar e remover palavras como favoritas.
4. Acesso a uma lista de palavras previamente pesquisadas.
5. Caching de resultados das requisições à API para agilizar buscas repetidas.
6. Seguir o wireframe para a página de listagem dos dados.

## Funcionalidade Diferencial

Implementação de autenticação com usuário e senha, associando favoritos e histórico ao ID do usuário e salvando essas informações em um banco de dados local ou remoto (Firebase, por exemplo).
